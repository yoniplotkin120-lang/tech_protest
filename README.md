# Tech Protest Data Analysis

This repository contains the Python workflow for preparing and analyzing survey data from the Israeli High-Tech Protest study.

The project combines two survey samples:

* Meta recruitment sample
* Panel recruitment sample

The analysis is organized as a sequence of Jupyter notebooks so that each stage of data preparation and analysis is transparent and reproducible.
### Planned notebooks

* `02_clean_raw_data.ipynb` — technical cleaning and validation of the merged raw data
* `03_construct_variables.ipynb` — recoding and construction of analytical variables
* `04_exploratory_data_analysis.ipynb` — descriptive statistics and exploratory analysis
* later notebooks — measurement models and preregistered structural analyses

## Data availability

The survey datasets are not included in this public repository because they may contain sensitive or identifying information.
